## Installation

```bash
$ npm install
```

## Docker Use
### Docker Build
```bash
$ docker-compose up --build
```

### Docker Run
```bash
$ docker-compose up
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
